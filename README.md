## 📌 Sobre o projeto

Este projeto foi desenvolvido com base em um modelo de referência, porém sem copiar o código. A proposta era recriar o layout utilizando apenas HTML e CSS, aplicando o conhecimento adquirido durante os estudos.

## 💡 Desafio

Durante o desenvolvimento, enfrentei algumas dificuldades principalmente com:

* Posicionamento de elementos
* Uso de background com imagem
* Organização do layout

Mesmo assim, optei por não copiar o código do modelo e buscar soluções por conta própria.

## 🧠 Aprendizados

Esse projeto foi muito importante para minha evolução como desenvolvedora, pois consegui:

* Entender melhor como funciona o CSS na prática
* Trabalhar com flexbox para alinhamento
* Aprender a lidar com erros e persistir até encontrar a solução
* Desenvolver autonomia na construção de layouts

## 🎨 Resultado

O resultado final ficou diferente do modelo original, mas foi intencional. Ajustei o layout de acordo com meu gosto e visão, criando uma versão única do projeto.

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3

Esse projeto representa um passo importante na minha jornada como desenvolvedora Front-End, mostrando minha capacidade de aprender, me adaptar e construir interfaces do zero.
